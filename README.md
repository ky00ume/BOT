# BOT

A Discord bot built with [discord.py](https://discordpy.readthedocs.io/).

## Setup

1. **Install dependencies**

   ```bash
   pip install -r requirements.txt
   ```

2. **Configure environment variables**

   ```bash
   cp .env.example .env
   # Edit .env and fill in your values
   ```

3. **Run the bot**

   ```bash
   python main.py
   ```
