# 폰트 번들 디렉토리

bg3_renderer.py가 이 디렉토리에서 폰트를 우선 탐색합니다.

## 권장 폰트 파일
- `NotoSerifCJK-Regular.ttc` (또는 .otf) — 한글 기본
- `NotoSerifCJK-Bold.ttc` (또는 .otf) — 한글 굵게
- `Lora-Variable.ttf` — 영문 세리프 (선택)

## 설치 방법
1. [Google Noto Fonts](https://fonts.google.com/noto) 에서 Noto Serif CJK KR 다운로드
2. 이 디렉토리에 .ttc 또는 .otf 파일 복사
3. 봇 재시작 시 자동 인식
